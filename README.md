# 🎓 Student Performance Dashboard

A customized admin dashboard built on the Mazer Bootstrap 5 template.
Displays student performance data loaded dynamically from a JSON file.

## Features
- 4 dynamic stat cards (Total Students, Avg Score, Top Performers, Attendance)
- Line chart showing monthly score trends using ApexCharts
- Student records table with live search filter
- Progress bars showing score visually
- Color coded status badges
- Responsive layout for mobile and desktop
- Custom EduDash branding

## How to Run
1. Clone this repository
2. Open the dist folder in VS Code
3. Right-click index.html and select Open with Live Server
4. No build step or npm required

## Files Changed
- dist/index.html — complete dashboard layout rebuilt
- dist/data/students.json — student data file created
- dist/assets/static/js/dashboard-custom.js — data loading and rendering logic

## Tech Used
Bootstrap 5, ApexCharts, Vanilla JavaScript ES6, HTML5, CSS3
